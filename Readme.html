Copy this files to the Libraries folder from your Arduino IDE or Atmel Studio IDE.

To use it in Atmel Studio, include the .cpp file, instead of the .h file.

Arduino IDE:
#include <RemoteControls.h>

Atmel Studio:
#include <RemoteControls.cpp>
